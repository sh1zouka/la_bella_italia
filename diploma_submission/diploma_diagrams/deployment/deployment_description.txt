============================================================
НАЗВАНИЕ ДИАГРАММЫ: Диаграмма развёртывания
    (Deployment Diagram)
============================================================

ОПИСАНИЕ:
Диаграмма развёртывания отображает физическую архитектуру
системы доставки «La Bella Italia» — распределение
программных компонентов по вычислительным узлам.

УЗЛЫ СИСТЕМЫ:

1. КЛИЕНТСКИЙ КОМПЬЮТЕР (Пользователь)
   Платформа: Windows / macOS / Linux
   Программное обеспечение:
   - Современный браузер (Chrome, Edge, Firefox)
   - HTML/CSS/JS — фронтенд приложения
   - localStorage — хранение корзины и данных сессии
   Связь: HTTP/1.1 через REST API (порт 3000)
   Примечание: не требует установки дополнительного ПО

2. СЕРВЕР ПРИЛОЖЕНИЙ (Node.js)
   Среда выполнения: Node.js 18+
   Компоненты:
   - Express.js — веб-сервер и маршрутизация
   - server.js — главный модуль с API-эндпоинтами
   - database.js — инициализация БД и NeDB-обёртка
   - Модели: User.js, MenuItem.js, Order.js
   - Статическая раздача файлов из папки frontend/
   Режимы работы:
   - DB_ENGINE=postgres — работа с PostgreSQL
   - DB_ENGINE=nedb — работа со встроенной БД
     (не требует установки PostgreSQL)
   Порт: 3000

3. СЕРВЕР БАЗЫ ДАННЫХ (PostgreSQL 16)
   Программное обеспечение:
   - PostgreSQL 16 — реляционная СУБД
   - База данных: italian_delivery
   - Таблицы: users, menu_items, orders
   - JSONB-поля для гибкого хранения данных
   - Индексы для оптимизации запросов
   Связь: TCP/IP (порт 5432)

СВЯЗИ МЕЖДУ УЗЛАМИ:
  Клиент (браузер) ──HTTP 3000──→ Сервер (Node.js)
  Сервер (Node.js)  ──TCP 5432──→ БД (PostgreSQL)

ОСОБЕННОСТИ РАЗВЁРТЫВАНИЯ:
- Система может работать в двух режимах: с PostgreSQL
  (продакшн) и со встроенной NeDB (демо/разработка)
- Фронтенд раздаётся сервером как статика — не требуется
  отдельный веб-сервер (nginx, Apache)
- Все компоненты могут быть развёрнуты на одной машине
  для демонстрации или на разных серверах для продакшна
- Для запуска достаточно Node.js — не требуется
  компиляция, сборка или установка дополнительных
  инструментов
