============================================================
НАЗВАНИЕ ДИАГРАММЫ: Диаграмма вариантов использования
    (Use Case Diagram)
============================================================

ОПИСАНИЕ:
Диаграмма вариантов использования описывает функциональные
требования к системе доставки «La Bella Italia» с точки
зрения трёх категорий пользователей (акторов).

АКТОРЫ СИСТЕМЫ:
1. Гость (неавторизованный пользователь)
   - Может просматривать меню и искать блюда
   - Может зарегистрироваться или войти в систему
   - Не имеет доступа к корзине и заказам

2. Клиент (зарегистрированный пользователь)
   - Имеет все возможности гостя
   - Может управлять корзиной (добавлять/удалять блюда)
   - Может оформлять заказы с использованием бонусной системы
   - Может просматривать историю заказов и свой профиль
   - Может отслеживать статус текущего заказа

3. Администратор
   - Имеет все возможности клиента
   - Может управлять меню (добавлять и удалять блюда)
   - Может управлять заказами (изменять статус)
   - Может просматривать список всех пользователей
   - Имеет доступ к статистике (выручка, средний чек,
     количество заказов)

ВАРИАНТЫ ИСПОЛЬЗОВАНИЯ (всего 13):
  UC-1  Просмотр меню                      — все акторы
  UC-2  Поиск и фильтрация блюд            — все акторы
  UC-3  Регистрация нового аккаунта        — гость
  UC-4  Вход в систему                     — гость
  UC-5  Управление корзиной                — клиент
  UC-6  Оформление заказа                  — клиент
  UC-7  Просмотр истории заказов           — клиент
  UC-8  Просмотр профиля и бонусов         — клиент
  UC-9  Отслеживание статуса заказа        — клиент
  UC-10 Управление меню                    — администратор
  UC-11 Управление заказами                — администратор
  UC-12 Управление пользователями          — администратор
  UC-13 Просмотр статистики                — администратор

ОТНОШЕНИЯ:
- UC-5 (корзина) → UC-6 (заказ): «include» —
  оформление заказа обязательно включает корзину
- UC-3 (регистрация) → UC-4 (вход): «extend» —
  после регистрации пользователь автоматически входит
